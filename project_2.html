Vector Art Portrait Making, Poster Design!

I designed a self-portrait of myself using Adobe Illustrator, and used different web boswsers (e.g Github Glich Tool and Giphy)

For this assignment, I had to take a photo of myself and add it into Adobe Illustrator and making it into a vector portrait poster. Myself-portrait was constructed by using the pen tool and adding many layers of color and shadow to my portrait.

The second part of this assignment was to add a logo of "Seattle Art Museum" (SAM) and add it to my portrait. In order to add the logo into the portrait, I had to trace the logo by using the pen tool and copy and paste the trace logo onto my portrait. The last part of this assignment was to uses different websites like (e.g Github Glich Tool and Giphy) to have these crazy effects to my portrait.
